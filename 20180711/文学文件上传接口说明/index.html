文学文件上传接口说明

签名：接口参数中注明需要签名的字段按字母升序拼接，然后拼上约定的key，最后再拼上接口名，进行md5加密并将结果转大写。
签名举例：
签名key为：123456789
参数为： phoneNo=136111111&t=124000000
接口名为：getphonecode
sign=md5(136111111124000000123456789 getphonecode). toUpperCase()

通用参数

参数名 描述 必填 签名
userid 用户id 是 是
sessionid 登录验证串 是 是
suffix 文件后缀 是 是
sign 签名 是 否

通用返回,json格式：

{
“code”: 1, //结果码，1成功，2参数错误，3文件类型错误，99失败
“errorMsg”: “success”,//错误说明，需要urldecode
“fileName”: “8989485ab893cde.jpg”//文件名，仅在code为1时有效
}

1.上传头像

参数名 描述 必填 签名
type 文件类型，上传头像，取值为headimg 是 是

2.上传普通图片

参数名 描述 必填 签名
type 文件类型，上传头像，取值为commonimg 是 是

3.上传书内容

参数名 描述 必填 签名
type 文件类型，上传头像，取值为bookfile 是 是
bookid 书编号 是 是
chapterid 章节编号 是 是

4.上传书封面

参数名 描述 必填 签名
type 文件类型，上传头像，取值为bookcover 是 是
bookid 书编号 是 是
chapterid 章节编号 是 是
